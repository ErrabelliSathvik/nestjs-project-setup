# NestJS Project Setup

## Features
- NestJS backend
- PostgreSQL (Neon)
- CRUD APIs
- Validation (DTO)
- Environment variables (.env)
- Modular structure

## Setup

1. Install dependencies:
   npm install

2. Create .env file:
   DATABASE_URL=your_database_url

3. Run:
   npm run start

## APIs
- POST /users
- GET /users
- PUT /users/:id
- DELETE /users/:id
